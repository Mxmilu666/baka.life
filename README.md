# baka.life Free Domains

[中文说明](README.zh-CN.md)

`baka.life` provides free subdomains for research, education, labs, personal learning, open-source experiments, and non-commercial technical projects.

This repository is used to request and manage DNS records. The automation code is supporting infrastructure; the main purpose is domain registration and DNS review.

The `baka.life` domain is supported by NYANLOLI LTD ([nyanloli.com](https://nyanloli.com)), while this free subdomain registry is operated by the community.

## Who Can Apply

You may request a subdomain for:

- Research projects
- Education and teaching
- Student labs
- Open-source projects
- Personal learning environments
- Non-commercial technical experiments

## Not Allowed

Do not use `baka.life` subdomains for:

- Phishing, malware, spam, or abuse
- Illegal content or illegal services
- Impersonation or misleading services
- Commercial resale of subdomains
- High-risk production services without prior approval

Maintainers may reject or remove records that violate these rules.

## How To Request A Domain

Submit a Pull Request to request or update a subdomain. First-time registration requires maintainer review.

For file format, DNS record format, validation rules, and PR workflow details, read:
