# baka.life 免费域名

[English](README.md)

`baka.life` 提供免费的子域名，主要用于科研、教育、实验室、个人学习、开源实验和非商业技术项目。

这个仓库用于申请和管理 DNS 记录。仓库里的自动化代码只是辅助设施；这里的主要用途是域名注册和 DNS 记录审核。

`baka.life` 域名由 NYANLOLI LTD（[nyanloli.com](https://nyanloli.com)）支撑，但这个免费子域名注册表由社区运营。

## 谁可以申请

你可以为以下用途申请子域名：

- 科研项目
- 教学与教育用途
- 学生实验室
- 开源项目
- 个人学习环境
- 非商业技术实验

## 不允许的用途

请不要将 `baka.life` 子域名用于：

- 钓鱼、恶意软件、垃圾信息或滥用
- 非法内容或非法服务
- 冒充他人或误导性服务
- 转售子域名
- 未经事先确认的高风险生产服务

维护者可以拒绝或移除违反规则的记录。

## 如何申请域名

提交 Pull Request 申请或更新子域名。初次注册需要维护者 review。
